# capacity-planning-simulation
Built a Monte Carlo simulation integrated with Excel Solver to determine the optimal vaccine production capacity across two manufacturing plants (Toronto & Vancouver) serving three national distributors over 6 years.
Tools: Excel VBA, Macros, Excel Solver
Key features: Automated simulation engine, 9 capacity plans evaluated, live UI feedback, histogram outputs
Outcome: Recommended Plan 8 (Toronto 900 / Vancouver 700) as the highest expected-profit strategy
